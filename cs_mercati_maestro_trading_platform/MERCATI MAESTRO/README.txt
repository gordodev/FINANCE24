# CS Mercati Maestro

## Application Purpose
CS Mercati Maestro is a trading application front end designed to provide comprehensive market data, position management, market surveillance, and compliance functionalities.

## Directory Structure
- **bin/**: Binaries and executables
- **config/**: Configuration files
  - app_config.yaml: Main application configuration
  - database_config.yaml: Database configuration
- **data/**: Data files
  - historical_data/: Historical market data
  - real_time_data/: Real-time market data
  - user_data/: User-specific data files
- **cache/**: Cache files
  - market_data_cache/: Cached market data
  - user_cache/: Cached user data
- **lib/**: Libraries
  - external/: External libraries
  - custom/: Custom libraries
- **logs/**: Log files
  - app.log: Main application log
  - error.log: Error log
  - access.log: Access log
- **src/**: Source code
  - components/: Front-end components
  - services/: Services and business logic
  - utils/: Utility functions
  - main.py: Main entry point of the application
- **tests/**: Test files
  - unit_tests/: Unit tests
  - integration_tests/: Integration tests
  - test_config.yaml: Test configuration
- **docs/**: Documentation
  - API_Documentation.txt: API documentation
  - User_Guide.txt: User guide
- **README.txt**: Project overview and setup instructions
